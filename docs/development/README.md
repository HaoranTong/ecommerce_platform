1. **[开发规范](standards.md)** - 了解代码规范和开发原则
2. **[开发工具](tools.md)** - 配置开发环境和工具
3. **[工作流程](workflow.md)** - 熟悉开发协作流程
4. **[测试指南](testing.md)** - 掌握测试策略和方法
5. **[文档规范](documentation-standards.md)** - 学习文档编写要求
