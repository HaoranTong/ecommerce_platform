## ⚡ 核心功能

- **会员等级管理** - 基于消费金额的自动等级晋升
- **积分系统** - 积分获得、使用、过期和冻结管理
- **会员档案** - 个人信息管理和偏好设置
- **权益体系** - 等级权益和个性化服务

## 🚀 快速开始

### API入口
- **基础路径**: `/api/v1/member-system`
- **认证方式**: JWT Bearer Token
- **数据格式**: JSON

### 关键接口
- `GET /profile` - 获取会员信息
- `POST /points/earn` - 积分获取
- `POST /points/use` - 积分使用
- `GET /levels` - 等级列表
