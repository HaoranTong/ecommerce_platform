# 业务服务层

业务逻辑服务模块，封装复杂的业务处理流程和数据操作。

## 📁 目录结构

```
services/
├── inventory.py           # 库存管理业务逻辑
└── __init__.py            # 模块初始化
```

## 🔧 服务说明

| 服务文件 | 主要功能 | 职责范围 |
|---------|---------|---------|
| **inventory.py** | 库存管理服务 | 库存查询、更新、预占、释放 |

## 🏗️ 服务层架构

```
API层 → 服务层 → 数据层
     ↓
   业务逻辑封装
   事务处理
   数据验证
   错误处理
```
